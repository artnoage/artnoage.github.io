   Meta-learning techniques for few-shot behavior prediction in multi-agent Ad-Hoc cooperation. The game of Hanabi

Recently Google released a new testbed for RL. The game of Hanabi has many new complexities that older testbed games like Chess and Go do not exhibit. Apart from a quite big state space, the game involves partial information and cooperation between agents. This cooperation is extremely difficult to model because communication between agent can contain meta-information. Although agents that are trained together can achieve high scores, when aggents with different learning experience meet, they exhibit poor performance. The goal is to apply meta-learning ideas to train the agents, so they will be able to adapt to new enviroments (different set of cooperative agents).

   Creating pool of agents for board games using evolutionary methods.

The goal is to create a list of acting rules that one can use to play a board game (most likely Hanabi), and then use evolutionary techniques to create agents that perform well in the game, by applying part of these rules. Statistical analysis can further be made to characterize diversity of the agents and fitness. The project is a bit tedious, but it is straightforward.

   Creating a Neural network which analyzes the decision process of humans reacting on a random dot stimuli.

There are several analytical models that interpret how humans process the uncertainty created by a random dot kinematographic (RDK), and produce a number that captures the "belief" of a subject over the hidden-state of the dots. These models operate on various assumptions and there is no obvious way to test if they actually behave the way they claim they do. However, there is a way to check if these belief satisfy some properties that control theory/decision making theory dictates. The goal is the creation of a Machine learning model that gets some input a RDK and the time of an action and spilts out a belief over the hidden-state of the dots. The model will be trained on part of our data and tested on the rest. Then its predicting power will be compared with one of the analytical models that we have at hand.
